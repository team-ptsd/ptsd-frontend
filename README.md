# PTSD - Frontend

## Installation
1. Install [Node.js](https://nodejs.org/en/download/)
2. Install [Git](https://git-scm.com/downloads)
3. Clone the repository
```bash
git clone <repo> 
```
4. Navigate to the repository
```bash
cd <repo>
```
5. Install dependencies
```bash
npm install
```
6. Run the application
```bash
npm run dev
```
